## Installation
The installation requires docker and docker-compose.

Clone the repository
```
git clone https://github.com/sidharthramesh/medblocks.git
cd medblocks

```
Build the docker images
```
docker-compose build
```

## Usage
To bring up containers
```
docker-compose up
```
